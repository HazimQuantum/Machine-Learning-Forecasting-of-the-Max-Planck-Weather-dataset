# Machine-Learning-Forecasting-of-the-Max-Planck-Weather-dataset

This dataset contains 14 different features such as air temperature, atmospheric pressure, and humidity. These were collected every 10 minutes, beginning in 2003. For efficiency, you will use only the data collected between 2009 and 2016.
Exploratory data analysis was performed, a distribution analysis of the wind speed as its the target variable. finally a ML Forecasting pipeline was created using Random Forrest Regression Algorithm.
